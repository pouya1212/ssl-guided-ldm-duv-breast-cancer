# ssl-guided-ldm-duv-breast-cancer
Official implementation of "Self-Learned Representation-Guided Latent Diffusion Model for Breast Cancer Classification in Deep Ultraviolet Whole Surface Images". The method generates synthetic DUV-FSM patches via SSL-guided latent diffusion and improves ViT-based classification performance.


## Installation

Clone repository

git clone https://github.com/pouya12/ssl-guided-ldm-duv-breast-cancer.git

cd ssl-guided-ldm-duv-breast-cancer

Install dependencies

pip install -r requirements.txt
